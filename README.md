## 项目简介

本项目是基于Spring Boot开发的民宿管理系统，旨在为用户提供一个便捷、高效、易用的民宿业务处理平台。系统涵盖了民宿预订、房间管理、客户管理、订单管理、财务管理等功能模块，满足民宿日常运营需求。

## 项目功能

1. **用户管理**：实现对系统用户的注册、登录、信息修改、权限控制等功能。
2. **房间管理**：包括房间类型、房间设施、房间价格、房间状态等信息的维护。
3. **预订管理**：实现客户预订房间、预订查询、预订修改、预订取消等功能。
4. **订单管理**：对客户订单进行管理，包括订单生成、订单查询、订单修改、订单取消等操作。
5. **客户管理**：维护客户基本信息，包括客户姓名、联系方式、会员等级等。
6. **财务管理**：实现订单支付、退款、发票、收支统计等财务功能。
7. **系统设置**：包括系统参数配置、权限管理、日志管理等。

## 技术栈

1. **前端技术**：
   - **HTML5**：用于构建网页结构。
   - **CSS3**：美化页面样式。
   - **JavaScript**：实现页面交互。
   - **Vue.js**：构建前端SPA应用。
   - **Element UI**：基于Vue的前端组件库，提高开发效率。

2. **后端技术**：
   - **Spring Boot**：构建后端应用，简化开发流程。
   - **Spring Security**：实现用户权限控制。
   - **MyBatis**：数据持久化框架，简化数据库操作。
   - **MySQL**：关系型数据库，存储数据。
   - **JWT**：实现用户认证与授权。

3. **开发工具与环境**：
   - **IntelliJ IDEA**：集成开发环境。
   - **Git**：版本控制系统。
   - **Maven**：项目构建工具。
   - **Docker**：容器化部署应用。

通过以上技术栈，本项目为用户提供了一个功能完善、易于扩展的民宿管理系统。系统采用前后端分离的设计，方便维护与升级。后端提供RESTful API，供前端调用，实现业务逻辑。整体架构清晰，易于二次开发。
